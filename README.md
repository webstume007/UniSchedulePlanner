# IUB AI Department Timetable Generator 📅

An automated, constraint-based scheduling system designed specifically for the Artificial Intelligence department at The Islamia University of Bahawalpur.

## Features
- **Constraint Programming Engine:** Built with Google OR-Tools to guarantee zero class clashes.
- **Dynamic Rules:** Handles Jumma breaks, combined lab sessions, and variable credit hour blocks.
- **Capacity Matching:** Ensures section strengths do not exceed physical room capacities.
- **Interactive Dashboard:** Built with Streamlit for easy CRUD operations and timetable visualization.
